Projeto Pessoal

Este dashboard foi desenvolvido para oferecer uma visão completa e detalhada das finanças pessoais. Com uma interface moderna e intuitiva, ele permite o acompanhamento em tempo real de receitas, despesas, investimentos e metas financeiras.

O projeto foi construído do zero utilizando tecnologias web modernas, com foco em performance, usabilidade e um design dark premium que torna a análise de dados uma experiência agradável. Os gráficos interativos permitem filtrar dados por período, categoria e tipo de transação.
